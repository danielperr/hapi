## Installing With Git

:::note
Installing Git is not required, but it's highly encouraged if you are willing to
contribute to the project or make any changes.

If, however, you do not want to install Git you can
[download the source files instead](https://github.com/danielperr/hapi/archive/master.zip)
and [skip this part](#installing-npm).
:::


Check if you already have Git installed:

```bash
git --version
```

[Homebrew](https://brew.sh/) is a helpful package manager for macOS.

If you don't have it, install it using the following command:


```bash
brew install git
```

### Configuring Git for commits

Configure your Git username and email using the following commands.
These details will be associated with any commits that you create:

```bash
git config --global user.name "Firstname Lastname"
git config --global user.email "your@email.com"
```

### Downloading the project from GitHub

Finally, clone (download) the project's repository from GitHub:

```bash
git clone https://github.com/danielperr/hapi
```

This will create a folder at your current location containing the project's files.

## Installing NPM

This project is built with [NPM](npm), so you will need to install it.

Check if you already have NPM installed:

```bash
npm -v
```

NPM is shipped with Node, a JavaScript runtime engine.

## Installing depencencies

After [cloning with Git](#downloading-the-project-from-github) (or just downloading the files),
navigate to the project's root directory:

```bash
cd hapi
```

Every package has its own dependencies (NPM packages that it depends on).

There are 3 packages, located at `packages/` (see [Project Structure](project-structure)):

* `hapi-activity`
* `hapi-creator`
* `hapi-docs`

To install packages for `hapi-creator` for example, first navigate to it:

```bash
cd packages/hapi-creator
```

Then use NPM to install its dependencies:

```bash
npm install
```

This will create the folder `node_modules`, containing the required packages.
